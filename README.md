This project is a web-based interactive demo of **Animate.css**, a popular CSS animation library. It allows users to preview, apply, and copy CSS animations on a text element in real-time. The main purpose is to provide a visual and interactive way to explore different CSS animations without writing code from scratch.

## âœ… Features
* Apply **Animate.css animations** to the main title (`#title`) with a single click
* Animations are **categorized**:
  * Attention Seekers
  * Back Entrances/Exits
  * Bouncing Entrances/Exits
  * Fading Entrances/Exits
  * Flippers
  * LightSpeed
  * Rotating Entrances
* Copy animation class names to clipboard for easy reuse
* **Dark/Light mode toggle** with smooth color transitions
* Clean and modern **UI design** using Google Fonts (`Work Sans`)
* **Hover effects**: Copy icons appear only when hovering over animation items
* Smooth transitions for color and background changes on theme switch
